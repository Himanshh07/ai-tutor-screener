# AI Tutor Screener

A smart AI interviewer that helps evaluate tutors by having a real conversation and analyzing their teaching style.

## 🚀 What it does

This project simulates a short interview where the AI talks to a tutor and asks questions about their subject, teaching style, and how they handle students.

Based on the conversation, it gives feedback on:
- Clarity of explanation
- Teaching ability
- Communication skills
- Overall performance

## ✨ Features

- Voice-based interaction (no typing required)
- Natural conversation with AI
- Structured interview flow
- Instant feedback with scores and suggestions

## 🛠️ Tech Stack

- Next.js
- OpenAI API
- Tailwind CSS
